# Mochi-Link 简化指令快速参考

## 🚀 快速开始

### 1️⃣ 绑定服务器 (管理员)
```
mochi.bind.add <服务器ID>
```

### 2️⃣ 使用简化指令

---

## 📋 指令列表

| 指令 | 别名 | 权限 | 功能 |
|------|------|------|------|
| `添加白名单 <玩家名>` | `白名单添加` `wl` | Level 2 | 添加玩家到白名单 |
| `在线` | `online` `玩家` | Level 1 | 查看在线玩家和服务器状态 |

---

## 💡 使用示例

### 添加白名单
```
添加白名单 Steve
白名单添加 Alex
wl Notch
添加白名单 "Player Name"  # 玩家名包含空格时使用引号
```

**返回**:
```
✅ 已将 Steve 添加到服务器 生存服 的白名单
```

**⚠️ 重要提示**: 玩家名包含空格时，必须使用引号括起来！

---

## ⚠️ 常见错误

### 未绑定服务器
```
请先在群组中绑定服务器
用法: mochi.bind.add <serverId>
```

**解决**: 使用 `mochi.bind.add <服务器ID>` 绑定服务器

---

### 服务器离线
```
❌ 服务器 生存服 当前离线
```

**解决**: 等待服务器上线或检查服务器连接

---

### 权限不足
```
权限不足：需要受信任用户权限（等级 2）
```

**解决**: 联系管理员提升权限等级

---

## 🔑 权限等级

| 等级 | 名称 | 可用指令 |
|------|------|----------|
| 1 | 普通用户 | `在线` |
| 2 | 受信任用户 | `在线` `添加白名单` |
| 3 | 管理员 | 所有指令 + 绑定管理 |
| 4 | 超级管理员 | 所有指令 + 危险操作 |
